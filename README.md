# life-is-short

Ubuntu server initialization script for Ubuntu 24.04 and newer.

## Remote execution

Run the initializer directly from GitHub:

```bash
curl -fsSL https://raw.githubusercontent.com/0xshawn/life-is-short/main/ubuntu_init.sh | sudo bash
```

The script must run as root and exits on non-Ubuntu systems or Ubuntu versions older than 24.04.
